# Demo Script

## Length

60 to 90 seconds.

## Sequence

1. Show a Google Calendar event.
2. Say:

   ```text
   Alexa, ask nudge to sync my calendar
   ```

3. Show the Alexa Reminder that was created.
4. Move the Google Calendar event time.
5. Show that the reminder updates.
6. Show the Echo routine using:

   ```text
   ask nudge to run quiet sync
   ```

7. End on the architecture visual or a short product summary.

## Notes

- keep private calendar details out of frame
- use short neutral event titles
- avoid claiming zero-setup automation
- describe the product as automatic after one-time setup
